# 💊 Farmácia Boa Vida

Sistema Front-End que simula a interface de gerenciamento de uma farmácia fictícia chamada **Farmácia Boa Vida**.

O projeto permite o gerenciamento de categorias de produtos, oferecendo funcionalidades completas de CRUD (Create, Read, Update, Delete).

---

## 🚀 Tecnologias Utilizadas

- ⚛️ React
- 🟦 TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🔀 React Router DOM
- 🔔 React Toastify
- 🌐 Axios
- 🪟 Reactjs Popup

---

## 📌 Funcionalidades

O sistema permite:

- ✅ Listar categorias
- ➕ Criar novas categorias
- ✏️ Editar categorias existentes
- ❌ Remover categorias
- 🔔 Feedback visual com notificações (Toastify)
- 🪟 Modais interativos para ações (Popup)

---

## 🏗️ Estrutura do Projeto

O projeto foi desenvolvido utilizando:

- Componentização com React
- Tipagem estática com TypeScript
- Consumo de API utilizando Axios
- Gerenciamento de rotas com React Router DOM
- Estilização moderna e responsiva com Tailwind CSS
